# Chloe Oriotis — Portfolio Site

Personal portfolio site for Chloe Oriotis: Engineer, Storyteller, Strategist.

Built with Claude AI. Hosted on GitHub Pages.

## How to deploy

1. Create a new GitHub repository named exactly: `chloeoriotis.github.io`
2. Upload `index.html` and your resume PDF (`Chloe_Oriotis_Resume_2026.pdf`) to the root of the repo
3. Go to **Settings → Pages → Source** and set branch to `main` / root
4. Your site will be live at `https://chloeoriotis.github.io` within a few minutes

## To add your articles

When you're ready to publish, create a new HTML file for each piece (e.g. `improv.html`, `story-problem.html`) and update the links in `index.html` from `#` to the file path.

## To update the resume

Replace `Chloe_Oriotis_Resume_2026.pdf` with your updated file — keep the filename the same so no links break.
